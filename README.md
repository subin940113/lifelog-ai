# lifelog-ai

자연어 기반 개인 로그를 구조화하고, 주간 단위로 집계하여  
개인의 행동 패턴을 분석하는 **라이프로그 실험 프로젝트**입니다.

본 프로젝트는 **LLM 기반 구조화를 최종 목표**로 하지만,  
현재는 MVP 단계로서 **구조화 파이프라인과 집계 로직을 먼저 검증**하는 데 초점을 둡니다.

---

## 프로젝트 목적

- 자연어로 기록되는 개인 로그를 **구조화된 이벤트 모델**로 변환
- 구조화된 데이터를 기반으로 **주간 단위 행동 패턴 집계**
- LLM 연동 전 단계에서 **도메인 모델, 배치, 집계 구조를 선제적으로 검증**

---

## 핵심 개념

### RawLog
- 사용자가 입력하는 **자연어 기반 로그**
- 예:
    - "어제 밤 12시에 잠들어서 6시간 정도 잤다"
    - "점심에 샐러드를 먹었고 기분이 괜찮았다"

### StructuredEvent
- RawLog를 기반으로 생성되는 **구조화된 이벤트**
- 조회 API, 집계 배치의 **기준 데이터**
- 현재는 **가짜 구조화(Pseudo Structured)** 단계로 자동 생성됨

### AggregatedInsight
- StructuredEvent를 기준으로 생성되는 **주간 집계 결과**
- 개인의 행동 패턴을 요약한 결과 데이터

---

## 구조화 단계

### 현재 구조화 방식: Pseudo Structured (가짜 구조화)

> 현재 프로젝트의 구조화는 **의도적인 가짜 구조화 단계**입니다.

- RawLog 생성 시점에
    - 규칙 기반 / 임시 로직으로 StructuredEvent 자동 생성
- LLM 호출은 수행하지 않음
- 수동 입력이나 더미 데이터가 아닌 **실제 파이프라인 상 자동 생성**

```text
RawLog 생성
   ↓
(Pseudo Structuring Layer)
   ↓
StructuredEvent 생성
